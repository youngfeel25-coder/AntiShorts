# Suno AI Prompt Templates

## How to Use in Suno AI

Suno AI has two input fields:
1. **Style of Music** — Paste the Style Tags (comma-separated keywords, max ~120 chars)
2. **Lyrics** — Paste the full lyrics with section markers like `[Verse]`, `[Chorus]`, `[Bridge]`

For instrumental tracks, check "Instrumental" and leave Lyrics blank.

## Analysis-to-Prompt Conversion Checklist

When converting a new YouTube track analysis to a Suno prompt, verify these 6 points:

- [ ] BPM is explicitly stated as a number (e.g., `88 BPM`, not just "fast")
- [ ] Key is stated (e.g., `D minor`, `E minor`)
- [ ] Scale type is included if distinctive (e.g., `harmonic minor`, `phrygian`)
- [ ] Each instrument has a technique descriptor (e.g., `palm-mute guitar`, not just `guitar`)
- [ ] Vocal style is specific (e.g., `gritty baritone power vocal`, not just `male vocal`)
- [ ] At least one spatial/production keyword is included (e.g., `large hall reverb`, `wall of sound`)
